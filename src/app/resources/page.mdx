# Resources

Curated external tools and references to support your learning throughout the program. {{ className: 'lead' }}

## Pre-Work & Foundations {{ id: 'prework' }}

These platforms are good starting points for building fundamentals or filling gaps.

- **[Codecademy](https://www.codecademy.com)** — Interactive courses for HTML, CSS, JavaScript, React, and more. Good for structured, guided learning.
- **[Scrimba](https://scrimba.com)** — Video-based coding courses where you can edit code inside the video. Strong for React and frontend topics.
- **[IBM SkillsBuild](https://skillsbuild.org)** — Free learning paths with professional certifications. Good for supplemental credentials.
- **[The Odin Project](https://www.theodinproject.com)** — A comprehensive free curriculum covering the full stack. Good for independent learners who want depth.

## Documentation {{ id: 'docs' }}

Get comfortable reading official documentation early. It is a core professional skill.

- **[MDN Web Docs](https://developer.mozilla.org)** — The definitive reference for HTML, CSS, and JavaScript. When in doubt, check MDN.
- **[React Docs](https://react.dev)** — The official React documentation with interactive examples.
- **[Node.js Docs](https://nodejs.org/docs)** — Reference for Node.js built-in modules.
- **[Express Docs](https://expressjs.com)** — Official Express framework documentation.
- **[PostgreSQL Docs](https://www.postgresql.org/docs)** — SQL and PostgreSQL reference.

## Practice Platforms {{ id: 'practice' }}

For extra reps on coding problems.

- **[Codewars](https://www.codewars.com)** — Short, gamified problems. Start at 8 kyu and work up.
- **[LeetCode](https://leetcode.com)** — Larger library of problems. Focus on Easy first.
- **[Exercism](https://exercism.org)** — Practice with real mentor feedback. Good for fundamentals.

## Interview Prep {{ id: 'interview' }}

Start thinking about this before you need it.

- **[NeetCode](https://neetcode.io)** — Structured problem sets with clear video explanations. One of the best free resources for interview prep.
- **[Pramp](https://www.pramp.com)** — Free peer mock interviews. Valuable for getting comfortable with talking through code.
- **[interviewing.io](https://interviewing.io)** — Anonymous mock interviews with engineers from major companies.

## Tools {{ id: 'tools' }}

Tools that will make your work easier.

- **[Postman](https://www.postman.com)** — Test your API endpoints without writing frontend code. Essential for backend work.
- **[NotebookLM](https://notebooklm.google.com)** — Upload course materials and ask questions about them. Good for reviewing concepts.
- **[Excalidraw](https://excalidraw.com)** — Quick whiteboarding for sketching out system designs or data flows.
- **[GitHub Student Developer Pack](https://education.github.com/pack)** — Free access to dozens of professional tools for verified students.
