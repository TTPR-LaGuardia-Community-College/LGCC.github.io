# Morning Exercises

Daily warm-up prompts archived here so you can revisit them, spot patterns, and track your own growth. {{ className: 'lead' }}

## How to Use This Archive {{ id: 'how-to-use' }}

Morning exercises are short, focused problems designed to activate your thinking and reinforce recent concepts. They are not tests — they are practice.

Use this archive to:

- **Revisit exercises** you found difficult to build muscle memory
- **Compare your approach** to the hints and sample guidance
- **Spot patterns** in the kinds of problems that trip you up
- **Reflect** on how your thinking has changed since earlier in the program

Each exercise page includes the original prompt, hints, a sample approach, common mistakes, an optional stretch version, and a reflection question.

## Exercise Template {{ id: 'template' }}

Every morning exercise page follows this structure:

- **Title and date** — When it was used in class
- **Target skill** — What concept this exercise reinforces
- **Prompt** — The original problem statement
- **Hints** — Pointers to get unstuck without giving away the answer
- **Sample approach** — One way to think through the problem
- **Common mistakes** — What most people get wrong the first time
- **Stretch version** — A harder extension if you want more challenge
- **Reflection prompt** — A question to help you consolidate what you learned
